# Chinese EV Expansion Dashboard

## Overview

This project is an interactive competitive intelligence dashboard focused on Chinese electric vehicle manufacturers and their international expansion.

The dashboard combines vehicle specifications, battery technologies and market expansion data to compare brands and evaluate their competitive positioning.

Brands included:

- BYD
- NIO
- Zeekr
- XPeng
- Li Auto
- Xiaomi Auto
- Jaecoo

---

## Objectives

- Analyze Chinese EV brands and models
- Compare price and driving range
- Evaluate battery technologies
- Assess international expansion potential
- Build custom competitiveness indicators

---

## Technologies Used

- Python
- Pandas
- Streamlit
- Matplotlib

---

## Features

### Interactive Brand Filtering

Users can select individual brands through the sidebar.

### Key Metrics

The dashboard displays:

- Average price
- Average range
- Average battery capacity
- Average expansion score

### EV Model Database

Vehicle specifications include:

- Price
- Range
- Battery type
- Market
- Data quality status

### Expansion Analysis

Brands are ranked based on their international expansion score.

### Price vs Range Visualization

Scatter plot comparing vehicle range and price.

### Battery Analysis

Comparison of average driving range by battery technology.

### Competitiveness Ranking

Custom score based on:

- Driving range
- Expansion strength
- Vehicle price

## Data Sources

Vehicle information is manually curated from public sources including:

- EV Database
- Manufacturer websites
- Public specifications
- Industry reports

Range standards used in the project include:

- EV Database Real Range
- WLTP
- CLTC

Expansion scores and competitiveness scores are analytical indices developed for this project and are not official industry metrics.
